Git Commands

Commun used Git Commnads and what they do.

  • :pencil2: Git init
    • Initialize a local Git repository
    • syntaxis: git init
  • Git Add
    • Saves your changes in a file to the staging area.

    • git add [file name]]

  • Git Commit
    • Records a snapshot of the file permanently in the version history.

    • git commit -m 'comment explaining changes since last commit '

  • Git Push
    • Push changes to remote repository (remembered branch)

    • git pushvariableName branchName

  • Git Status
    • Check status (It will inform us of Gits thoughts and the state of our repository as seen by Git.) Lists all files, stating files that have changes to be committed, or files that need to be added to staging area.

    • git status
