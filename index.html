Creating

  • Git Init :point_right: Initialize a local Git repository
    • git init
  • Git Clone :point_right: Copies a repository from an existing URL.
    • git clone repo_url https://github.com/example/example.Git

Share & Update

  • Git Push :point_right: Push changes to remote repository (remembered branch)
    • git push variable Name branch Name
  • Git Pull :point_right: Update local repository to the newest commit.
    • git pull https://github.com/example/example.git
  • Git Reset [file] :point_right: unstage changes to a file while retaining the changes in your working directory
    • git reset
  • Git Add [file name] :point_right: Saves your changes in a file to the staging area.To add a single file to GitHub
    • git add [file name]

Stage

  • Git Commit :point_right: Records a snapshot of the file permanently in the version history.
    • git commit - m (comment explaining changes since last commit)
  • Git Status :point_right: Check status (It will inform us of Gits thoughts and the state of our repository as seen by Git.)
    • git status
  • Git Revert :point_right: Revert commmit changes.
    • git revert
  • Git Remove :point_right: Removes individual files or a collection of files from the staging area and the working directory of the current branch.
    • git rm File.txt

Merge & Review

  • Git Merge :point_right: The history of the specified branch is merged into the current branch with this command.
    • git merge branch Name
  • Git Config :point_right:
  • Sets the author name and email address to be used with the commit command.
    • 'git config --global user.name 'username' 'git config --global user.email 'user@email.com'>
  • Git Branch :point_right: lists all your branches an * will indicate your current branch
    • 'git branch
