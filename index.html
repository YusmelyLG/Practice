Creating

  • Git Init --> Initialize a local Git repository
    • syntax: git init
  • Git Clone --> Copies a repository from an existing URL.
    • syntax:git clone repo_url https://github.com/example/example.Git

Share & Update

  • Git Push --> Push changes to remote repository (remembered branch)
    • syntax:git push variable Name branch Name
  • Git Pull --> Update local repository to the newest commit.
    • syntax: git pull https://github.com/example/example.git
  • Git Reset [file] --> unstage changes to a file while retaining the changes in your working directory
    • syntax: git reset
  • Git Add [file name] --> Saves your changes in a file to the staging area.To add a single file to GitHub
    • syntax: git add [file name]

Stage

  • Git Commit --> Records a snapshot of the file permanently in the version history.
    • syntax: git commit - m (comment explaining changes since last commit)
  • Git Status --> Check status (It will inform us of Gits thoughts and the state of our repository as seen by Git.)
    • syntax: git status
  • Git Revert --> Revert commmit changes.
    • syntax:git revert
  • Git Remove --> Removes individual files or a collection of files from the staging area and the working directory of the current branch.
    • syntax:git rm File.txt

Merge & Review

  • Git Merge --> The history of the specified branch is merged into the current branch with this command.
    • syntax: git merge branch Name
  • Git Config -->
  • Sets the author name and email address to be used with the commit command.
    • syntax:'git config --global user.name 'username' syntax: 'git config --global user.email 'user@email.com'>
  • Git Branch --> lists all your branches an * will indicate your current branch
    • syntax:'git branch
